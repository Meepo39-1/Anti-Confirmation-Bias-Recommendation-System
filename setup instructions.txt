Python version: 3.10.0
download link: https://www.python.org/downloads/release/python-3100/


CREATE VIRTUAL ENVIRONMENT:
    open a terminal in project's folder location
    pip install virtualenv
    python -m venv env

ACTIVATE VIRTUAL ENVIRONMENT
    Windows:
        Set-ExecutionPolicy Unrestricted -Scope Process
        env/Scripts/activate

    On Unix or MacOS:
        source env/bin/activate


INSTALL DEPENDENCIES:
    pip install -r requirements.txt


TEST RECOMMENDATIONS:
    For content filtering type in terminal:
        python "data pipeline/data processing workflow/content_filtering_flow.py"

    For collaborative filtering, type:
        You can either:
            1. download git lfs: https://github.com/git-lfs/git-lfs?utm_source=gitlfs_site&utm_medium=source_link&utm_campaign=gitlfs
               and type: git lfs install in git bash
            2. Unzip any of raw_delta_log_submissions.zip or raw_delta_log_submissions.rar 
        
        open jupyter notebook "collaborative_filtering_flow.ipynb"
        and select the previously created (env) environment as the kernel

        Note: 
            For opening the notebook you can use:
                pip install notebook
                jupyter notebook collaborative_filtering_flow.ipynb
